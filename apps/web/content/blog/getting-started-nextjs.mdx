## Why Next.js?

Next.js is a React framework that makes building production-ready web apps easy. With the App Router, you get server components, streaming, and file-based routing out of the box.

## Quick Setup

```bash
npx create-next-app@latest my-app --typescript --tailwind
cd my-app && pnpm dev
```

## Key Features

- **Server Components** — render on the server, ship less JavaScript
- **File-based Routing** — create a file, get a route
- **Streaming** — progressive rendering for faster perceived load times

## What I Learned

Building my portfolio with Next.js taught me how to think in components and leverage static generation for blazing-fast page loads.
